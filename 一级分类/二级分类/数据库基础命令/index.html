数据库基础命令

发表于2023-03-04|更新于2023-03-04|一级分类二级分类
|阅读量:
  1. 进入 mysql : mysql -u+ username -p+ password;
  2. 退出 mysql : exit;
  3. 创建数据库: create database + databasename;
  4. 查看所有数据库: show databases;
  5. 进入数据库: use + batabasename;
  6. 查看数据库地址: show variables like ‘%datadir%’;
  7. 创建table表格: create table + tablename;
  8. 查看所有表格: show + tables;
  9. 查看指定表格: desc + tablename;
  10. 添加一栏表格内容: alter table tablename add column addname type;
  11. 修改一栏指定内容类型:alter table tablename modify column columnname type;
  12. 改变一栏指定内容名字与类型:alter table user change column upassword umima varchar(100);
  13. 删除一栏内容: alter table user drop column address;
  14. 重命名指定表格:alter table user rename to users;
  15. 向指定表格插入一栏指定内容: alter table users add column age int;
  16. 像指定表格输入值:insert into users values(1,’李四’,123456,18);
  17. 查询表格内容: select * from users;
  18. 向指定栏目插入内容:insert into users(id,uname,upassword) values(2,’王二’,111111);
  19. 删除复核条件的一栏:delete from users where id=3;
  20. 更换栏目内容:update users set uname=’李四’ where id=2;
  21. 转换成别名:select uname as ‘姓名’,upassword as ‘密码’ from users;
  22. 模糊查询:select * from users where uname like ‘李%’
  23. 最大,最小,平均,数量: select max(age),min(age),avg(age),count(*) from users;
  24. 按一个栏目的英文字母正序/逆序排序:select * from users order by uname desc/asc;
  25. 查看临时目录:select @@tmpdir;
  26. 数据存放的位置:select @@datadir;
  27. 数据库服务所在位置:select @@basedir;
  28. 查看版本号:select @@version;
  29. 查看当前用户名:select @@hostname;
  30. 查询ASCll码/返回字符:select char(‘a’);/select char(97);
  31. 查第几行数据:select * from user limit2,3;(2代表第二行数据,3代表该条位置后的3行数据);
  32. 获取用户名:select user();
  33. 获取当前版本号:select version();
  34. 获取当前数据库:select database();
  35. 休眠:select sleep();

以下取自:原文链接：https://blog.csdn.net/qq_44159028/article/details/114327303

version()： 查询数据库的版本
user()： 查询数据库的使用者
database()： 数据库
system_user()： 系统用户名
session_user()： 连接数据库的用户名
current_user： 当前用户名
load_file()： 读取本地文件
@@datadir： 读取数据库路径
@@basedir： mysql安装路径
@@version_complie_os： 查看操作系统
length() 返回字符串的长度
substring() 截取字符串
substr() 截取字符串
mid() 截取字符串
left() 从左侧开始取指定字符个数的字符串
concat() 没有分隔符的连接字符串
concat_ws() 含有分割符的连接字符串
group_conat() 连接一个组的字符串
ord() 返回ASCII 码
ascii() 返回ASCII 码
hex() 将字符串转换为十六进制
unhex() hex 的反向操作
md5() 返回MD5 值
sleep() 睡眠时间为指定的秒数
if(true,t,f) if 判断
